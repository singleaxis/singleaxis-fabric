# Support

## Open source (Apache 2.0)

The `singleaxis-fabric` OSS substrate is community-supported:

- **Bugs / feature requests:** open a GitHub issue.
- **Questions / usage:** GitHub Discussions.
- **Security vulnerabilities:** **do not** open a public issue — follow
  [`SECURITY.md`](SECURITY.md) for private disclosure.

Supported versions follow [`SECURITY.md`](SECURITY.md): until `1.0.0`,
the latest minor receives fixes. We aim to acknowledge issues within 3
business days (best-effort, not an SLA).
